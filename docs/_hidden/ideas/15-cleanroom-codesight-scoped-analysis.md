# Cleanroom Codesight With Scoped Analysis

- Build a cleanroom implementation of Codesight-style code analysis and report generation.
- Scope AST lookups and synthesis to a workspace, folder, or other explicit target.
- Generate different reports for different scopes without repo-wide noise.
- Success: users can ask for targeted analysis on one slice of a repo and get scope-specific output.
